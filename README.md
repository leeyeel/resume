[https://github.com/hijiangtao/resume](https://github.com/hijiangtao/resume)

## 使用方法

现在Ubuntu 20.04已经可以直接使用源安装了，不需要手动编译

```
sudo apt-get install texlive-full
```

安装完成后使用`make`即可生成简历
